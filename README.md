# help_SovaBot
Справочный Telegram-bot ver 1.0

1 Установка приложения "Справочный Telegram-бот"

1.1 Запустить файл ST_Teleram_Bot.exe.
1.2 Инсталлятор предлагает выбрать язык установки по умолчанию, будет стоять русский язык, нажимаем кнопку «ОК».
1.3 Далее инсталлятор предлагает выбрать путь, по которому приложение будет установлено, изменить путь можно нажав кнопку «Обзор…», нажимаем кнопку «Далее».
1.4 Инсталлятор предложит создать ярлык в меню «Пуск».
1.5 Следующим шагом инсталлятор предложит создать ярлык приложения на рабочем столе, рекомендуется отметить данный пункт, нажимаем кнопку «Далее».
1.6 Инсталлятор сообщит, что всё готово к установке и выведет дополнительные задачи, с которыми можно ознакомиться. Нажимаем на кнопку «Установить». 
1.7 После выполнения всех действий, приложение инсталлируется согласно заданному пути. Завершив инсталляцию приложения, инсталлятор сообщит о завершении установки. Нажмите «Готово».
1.8 Установите RhVoice-Anna

2 API-ключ:

2.1 Создание собственого Telegram-бота (2.1 по 2.1.6 пункты не обязательны).
2.1.1 Найти бота "BotFather" введя @botFather в текстовое поле "Поиск".
2.1.2 Подключаемся к даному боту. 
2.1.3 Прописывается команда «/newbot».
2.1.4 Вводите имя бота с окончание "bot".
2.1.5 После водите пользовательское имя, если имя существует "BotFather" отклонит его.
2.1.6 После выполнений всех действий создание и регистрация нового Telegram-бота завершится. «BotFather» вышлет API ключ, с помощью которого будет происходить подключение к справочному Telegram-боту.

3 Включение Telegram-бота

3.1 Нажать на ярлык "Sova-bot" от имени администратора.
3.2 Введите API ключ в текстовое поле "Ключ Bot Telegram" и нажмите кнопку "Включить бот".
3.3 Найти "Справочный Telegram-бот" можно найти введя ----- в текстовое поле "Поиск" в мессенджере Telegram.
3.4 Имя в мессенджере бота будет -------, подключаемся к даному боту.
3.5 Введите команду "/star" для ознакомление с текстовыми командами бота.

Внимание!!! В режиме "Поиск","Режим учебник", "Режим обучение" и за низкого соединение интернет могут происходить задержки
с отправлением справочной информации, а так же реакция на другие команды, поэтому дождитесь когда бот выведет аудиособщение,
это будет означать, то что бот вывел всю информацию.

4 Извлечение данных с web-сайтов

4.1 Введите ссылку на web-сайт в текстовое поле "URL-адреs".
4.1.1 Введите HTML-теги, если хотите извлечь информацию с этих HTML-тегов.
4.1.2 Нажмите на кнопку "Парсинг по тегам"
4.2 Нажмите на кнопку "Парсинг без тегов".
4.3 Извлеченые данные будут выведены в текстовое поле "Данные".

5 Кнопкой "Очистить текст от отступов" можно удалить лишние переносы в текстовом поле "Данные"

6 Запись в json-файлы

6.1 Запись в файл с именем "Explorer"
Например что-бы записать в json-файл "Explorer_C#" производится следующие действия:
- выбрать путь через поле "Путь к json-файлу" к json-файлу "Explorer_C#;
- заполнить текстовое поле "Имя json-объекта"; (это будет поисковым именем в разделе С#, в режим "Поиск");
- заполнить поле данные;
- нажать кнопку "Добавить запись в json-файл";
- нажать кнопку "Ок" в всплывающем окне.

6.2 Запись в файл с именем "Section"
Например что-бы записать в json-файл "Section_C#" производится следующие действия:
- выбрать путь через поле "Путь к json-файлу" к json-файлу "Section_C#.
- заполнить поле данные.
- нажать кнопку "Добавить запись в json-файл".
- нажать кнопку "Ок" в всплывающем окне.


6.3 Запись в файл с именем "Tutorial"
Например что-бы записать в json-файл "Tutorial_C#" производится следующие действия:
- выбрать путь через поле "Путь к json-файлу" к json-файлу "Section_C#.
- заполнить поле данные.
- нажать кнопку "Добавить запись в json-файл".
- нажать кнопку "Ок" в всплывающем окне.

7 Создание нового раздела в "Справочном Telegram-боте"

Например создание нового раздела для язык программирование 1C:
- создаются три файла с именами:"Explorer_1C","Section_1C","Tutorial_1C" в катологе "json" внутри директотрии приложения;
- в административном интерфейсе выбрете путь через поле "Путь к json-файлу" к json-файлу "Branch";
- в текстовое поле "Имя json-объекта" прописывается 1С;
- нажимается кнопка "Добавить запись в json-файл";
- нажать кнопку "Ок" в всплывающем окне;
- перезгрузить приложение накжав кнопку "Перезагрузка";
- в чате справочного Telegram-бота ввести команду "/начать".

В результате по мимо язык программирования:

- Python.
